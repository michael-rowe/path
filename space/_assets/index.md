# Protected assets

This folder holds images and screenshots used in the Path manual and interface. It is prefixed with `_` so SilverBullet excludes it from the main page list.

To reference an image from a manual page:

```
![[_assets/screenshot-name.png]]
```

Files here are served by SilverBullet like any other attachment. They will not appear in `Ctrl-K` search results or the page browser.
